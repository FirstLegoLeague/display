# rednblue theme

use it with the `css` module in the config like this:

        ...
        'css': {
            href: [
                'themes/rednblue/rednblue.css',
                // 'themes/rednblue/rednblue-mod-rtl.css'
            ]
        },
        ...

The rtl modifier css can be added for rtl text support.
